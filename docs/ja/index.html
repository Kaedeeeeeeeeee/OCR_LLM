よくある質問

短く、はっきり。

Cheese! OCR は無料ですか？
Cheese! OCR は Mac App Store で販売されている買い切り型の有料アプリです。サブスクリプションやアプリ内課金はありません。一度購入すれば、同じ Apple ID でサインインしているすべての Mac で利用できます。
オフラインで動作しますか？
はい。すべてのテキスト認識は Apple の Vision フレームワークを使ってデバイス上で完結します。アプリにはネットワーク権限が一切付与されていないため、データが外部に送信されることは技術的にもありません。
対応している言語は？
英語・中国語（簡体字）・日本語・韓国語に標準対応しています。一つの画像に複数言語（例：日本語と英語の混在）が含まれていても正しく認識されます。
macOS 標準のテキスト認識 (Live Text) とどう違いますか？
Live Text はプレビュー・写真・Safari など一部の Apple 純正アプリ内でしか使えません。Cheese! OCR はグローバルホットキーで、動画プレーヤー・会議アプリ・保護された PDF・リモートデスクトップなど、画面上のあらゆる場所のテキストを認識できます。
必要な macOS のバージョンは？
Cheese! OCR は macOS 13 Ventura 以降で動作します。Apple Silicon・Intel 両方の Mac に対応しています。
ホットキーは変更できますか？
はい。アプリの設定画面で任意のショートカットを録音できます。デフォルトは ⇧⌘E です。
スクリーンショットはどこかにアップロードされますか？
一切されません。スクリーンショットはメモリ内で Apple Vision により処理され、ディスクに保存されることも、ネットワーク経由で送信されることもありません。アプリにはネットワーク権限自体がありません。
