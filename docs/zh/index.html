常见问题

用最短的答案回答最常见的疑问。

Cheese! OCR 是免费的吗？
Cheese! OCR 是 Mac App Store 上的一次性付费应用，没有订阅，也没有内购。购买一次即可在所有使用同一 Apple ID 的 Mac 上使用。
可以离线使用吗？
可以。所有文字识别都在本地通过 Apple Vision 框架完成。应用完全没有网络权限，即便想联网也做不到，彻底杜绝数据外传。
支持哪些语言的文字识别？
开箱即用支持中文（简体）、英文、日文、韩文。同一个截图中混合多种语言（例如中英混排）也能正确识别。
和 macOS 自带的「实况文本」(Live Text) 有什么区别？
Live Text 只在少数系统应用（预览、照片、Safari）里可用。Cheese! OCR 通过全局快捷键，可以识别屏幕上任何位置的文字，包括视频播放器、会议软件、受保护的 PDF、远程桌面等任何场景。
需要什么版本的 macOS？
Cheese! OCR 支持 macOS 13 Ventura 及以上版本，Apple Silicon 与 Intel 芯片 Mac 都可以使用。
可以修改快捷键吗？
可以。在应用设置里打开快捷键录制，按下任意组合即可自定义。默认快捷键是 ⇧⌘E。
我的截图会被上传到服务器吗？
永远不会。截图仅在内存中通过 Apple Vision 处理，不会写入磁盘，更不会通过网络发送。应用没有任何网络权限。
